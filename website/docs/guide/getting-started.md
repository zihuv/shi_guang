# 快速开始

拾光是一款桌面端素材管理工具，适合整理灵感图、参考稿、项目素材和网页采集内容。

> 当前版本仍在开发阶段，建议先用一套可替换的测试素材跑通流程，再逐步扩大使用范围。

## 安装

从 [GitHub Releases](https://github.com/zihuv/shiguang/releases) 下载对应平台的安装包：

- Windows 10+
- macOS 10.15+
- Linux

## 首次启动

首次启动时，如果没有配置素材目录，应用会自动创建 `~/Pictures/shiguang` 作为默认目录。你也可以在设置里继续添加自己的本地目录。

## 导入素材

有四种方式把文件导入拾光：

- 点击顶部 **导入** 按钮批量选择本地文件
- 直接把文件或文件夹拖进应用窗口
- 从剪贴板粘贴图片
- 通过浏览器扩展采集网页图片

指定目标文件夹后导入，文件会进入该目录；否则进入默认目录 `~/Pictures/shiguang`。

## 整理素材

建议从这三件事开始：

1. 给常用类别建立文件夹层级
2. 给关键素材加标签，方便跨文件夹组织
3. 补上评分、备注和来源链接

这样以后找素材时，不必只靠文件名回忆。

## 查找素材

- **顶部搜索框**：按关键词检索
- **文件夹过滤**：缩小到某个目录范围
- **标签过滤**：按语义分类查找
- **类型过滤**：区分图片、视频、文档
- **颜色过滤**：按主色缩小结果

## 浏览器采集

安装浏览器扩展后，网页图片右键选择"发送给拾光"，图片会保存到"浏览器采集"文件夹。

详细安装和使用说明见：[浏览器扩展](./browser-extension)。

## 删除

支持两种模式：

- **回收站模式**：删除的文件进入回收站，可恢复
- **永久删除模式**：直接从文件系统删除

整理阶段建议先用回收站模式。
