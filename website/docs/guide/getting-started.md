# 快速开始

拾光是一款面向本地工作流的桌面素材管理工具，适合整理灵感图、参考稿、项目素材和网页采集内容。

> 当前版本仍在开发阶段，建议先用一套可替换的测试素材跑通流程，再逐步扩大使用范围。

## 1. 安装应用

从 [GitHub Releases](https://github.com/zihuv/shiguang/releases) 下载对应平台的安装包。

目前仓库说明中覆盖的平台包括：

- Windows 10 及以上
- macOS 10.15 及以上
- Linux

## 2. 第一次打开会发生什么

首次启动时，如果你还没有配置任何素材目录，应用会自动准备一个默认目录：

`~/Pictures/shiguang`

这个目录会作为第一套素材库使用。你也可以在应用的“设置 -> 素材目录”里继续添加自己的本地目录。

## 3. 添加或索引素材目录

在设置面板里可以完成两类操作：

- 添加素材目录：把现有本地文件夹纳入拾光管理
- 重建索引：在目录变化较大时重新扫描素材库

拾光不会为了“管理”而强制复制一份已存在的目录内容。对已有素材目录来说，它更像是在原地建立索引和辅助信息。

## 4. 导入素材的几种方式

你可以用下面几种方式把文件放进拾光的工作流里：

- 通过顶部“导入”按钮批量选择本地文件
- 直接把文件拖进应用
- 从剪贴板粘贴图片
- 从浏览器扩展采集网页图片

如果导入时没有指定目标文件夹，文件会进入默认导入目录 `~/Pictures/shiguang`。如果你是在某个文件夹节点上执行导入，文件会直接进入那个目标目录。

## 5. 整理素材

进入素材库后，推荐先做三件事：

1. 给常用类别建立文件夹层级
2. 给关键素材加标签，方便跨文件夹组织
3. 给重要文件补评分、备注和来源链接

这样做的意义是：以后找素材时，不必只靠文件名回忆。

## 6. 查找素材

拾光当前已经有一套比较完整的查找入口：

- 顶部搜索框：按关键词快速检索
- 文件夹过滤：缩小到某个目录范围
- 标签过滤：用语义分类找图
- 类型过滤：区分图片、视频、文档
- 颜色过滤：按主色快速缩小结果

如果你是做视觉参考、品牌板、UI 灵感库，这套组合会比只翻系统资源管理器高效得多。

## 7. 浏览器采集

如果你希望把网页图片也接入同一套素材库，可以使用仓库里的扩展：

1. 打开 `extensions/shiguang-collector`
2. 在 Chrome 或 Edge 进入 `chrome://extensions/`
3. 开启“开发者模式”
4. 选择“加载已解压的扩展程序”
5. 选择 `extensions/shiguang-collector` 目录

安装后，在网页中右键使用“采集”，图片会保存到拾光创建的“浏览器采集”文件夹。

## 8. 删除与安全

应用支持两种删除模式：

- 移入回收站
- 永久删除

如果你还在整理阶段，建议先使用回收站模式，避免一边归类一边误删。
