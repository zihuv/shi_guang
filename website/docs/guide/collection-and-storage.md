# 采集与数据

## 采集方式

素材可以通过四种方式进入拾光：

- 点击顶部 **导入** 按钮批量选择文件
- 直接从系统拖拽文件或文件夹到应用窗口
- 从剪贴板粘贴图片
- 通过浏览器扩展采集网页图片

## 默认素材目录

首次启动时，如果没有配置素材目录，应用会自动创建 `~/Pictures/shiguang` 作为默认目录。这也是未指定目标文件夹时的导入位置。

## 已有目录 vs 新导入文件

- **已有目录加入索引**：扫描并索引文件，不会额外复制一份数据
- **新文件导入**：文件复制到当前选中的文件夹，未指定时进入默认目录

## 浏览器采集

安装浏览器扩展后，网页图片通过右键菜单或拖拽发送到拾光，落入素材库的 **"浏览器采集"** 文件夹。这个文件夹在第一次采集时自动创建。

## 数据存放

### 素材目录内

每个素材目录下有一个 `.shiguang/` 隐藏目录，跟着素材目录走：

```
<素材目录>/.shiguang/
├── db/shiguang.db      # SQLite 数据库
└── thumbs/              # 缩略图缓存（WebP 格式）
```

- **数据库** — 存储文件夹结构、文件索引、标签、评分、备注、来源链接、颜色分布、设置等信息
- **缩略图缓存** — 自动生成的 768px WebP 缩略图，按文件内容哈希命名

### 应用数据目录

应用的配置信息（当前素材库路径、最近使用的库列表）保存在系统的应用数据目录中，迁移素材库时不需要额外处理。

## 删除

- **回收站模式**：删除的文件进入回收站，可以在回收站页面恢复。Cmd/Ctrl+Z 可以撤销最近一次删除操作（会话内有效）
- **永久删除**：直接从文件系统删除

整理阶段建议先用回收站模式。
