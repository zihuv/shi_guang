# 浏览器扩展

从网页采集图片到拾光素材库。

## 安装

扩展位于仓库 `extensions/shiguang-collector` 目录，需要以开发者模式加载：

1. 打开 Chrome 或 Edge，进入 `chrome://extensions/`
2. 开启右上角的 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `extensions/shiguang-collector` 目录

安装后，扩展图标会出现在工具栏。点击图标可以查看连接状态。

## 采集方式

### 右键菜单

在网页图片上点击右键 → 选择 **发送给拾光**。图片会自动发送到拾光，落入"浏览器采集"文件夹。

### Alt + 左键

按住 `Alt` 键，点击网页上的图片，直接发送。不需要打开右键菜单。

### 拖拽

把网页图片往页面底部拖，会出现一个悬浮 Dock，拖到上面松手即可发送。

## 小红书适配

在小红书浏览时，右键菜单中会多出一个"发送到拾光"选项，可以直接采集笔记中的图片。

## 前提条件

使用扩展前需要确保拾光桌面应用正在运行。扩展通过本地 `http://127.0.0.1:7845` 与应用通信。

## 采集的文件去哪了

采集的图片会进入当前素材库的 **"浏览器采集"** 文件夹，方便统一整理。
