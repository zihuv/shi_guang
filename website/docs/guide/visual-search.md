# 自然语言搜索

自然语言搜索会为图片建立本地视觉索引，让你可以用“红色包装海报”“极简登录页”“户外咖啡照片”这类描述找回素材。

这个功能在本机运行，不会把图片上传到外部服务。首次使用前需要准备视觉模型目录。

## 准备模型

当前推荐从 [模型发布页](https://github.com/zihuv/vl-embedding-test/releases) 下载已经整理好的模型包。

下载后按下面步骤处理：

1. 解压模型包到一个固定位置，例如 `~/Documents/shiguang-models/fgclip2_flat`
2. 打开拾光的“设置 -> AI -> 本地自然语言搜索”
3. 在“模型目录”里选择解压后的模型文件夹
4. 点击“校验目录”，看到模型状态可用后再启用自然语言搜索

建议不要把模型目录放进临时下载目录。后续应用会记住这个路径，如果目录被移动或删除，需要重新选择。

## 自动识别

启动时，拾光会尝试识别一些常见模型目录：

- 当前工作目录下的 `models/fgclip2_flat`
- 当前工作目录下的 `models/chinese_clip_flat`
- 开发调试目录 `.debug-models/fgclip2_flat`
- 开发调试目录 `.debug-models/chinese_clip_flat`

如果模型已经放在这些位置，设置页会自动填入可用路径。没有自动识别到时，手动选择目录即可。

## 建立索引

模型校验通过后，可以在设置页点击“处理未索引图片”。首次处理会花一些时间，后续新增图片只需要处理增量内容。

如果开启“导入后自动建立视觉索引”，新导入的图片会在后台排队处理。

## 使用建议

自然语言搜索适合查找视觉特征、画面内容和风格相近的图片。文件名、标签、颜色筛选仍然有价值，建议和自然语言搜索一起使用。
