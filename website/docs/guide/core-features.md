# 核心能力

拾光围绕素材管理的四个环节：收集、整理、筛选、找回。

## 多目录索引

把多个本地目录加入素材库，在同一个界面上浏览和管理。适合：

- 项目素材散落在不同硬盘或工作目录
- 下载目录里临时收了一堆图，后续需要归档
- 想把项目素材和长期灵感库放到同一套视图里

## 导入方式

- 批量文件导入
- 拖拽导入
- 剪贴板粘贴
- 浏览器扩展采集

不需要先整理再导入，可以先低成本把素材聚合起来。

## 文件夹与标签

文件夹表达物理归属，标签表达语义。两者并行：

- 文件夹按项目、来源或时间分层
- 标签标记风格、题材、颜色倾向、可复用场景

## 元数据

给素材补充额外信息：标签、评分、备注、来源链接。以后回看时，仍然记得它为什么被保存。

## 搜索与筛选

素材增多后，关键不是"继续收"，而是"能不能在一分钟内找回"。组合使用的查找维度：

- 关键词
- 文件夹
- 标签
- 文件类型
- 主色

## 后台导入

批量导入任务在后台执行，显示进度。导入过程中可以继续浏览和操作素材库，不需要等待。

## 回收站

两种删除模式：回收站（可恢复）和永久删除。大规模清理时建议先开启回收站模式。
