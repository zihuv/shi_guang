# 筛选与排序

筛选面板位于素材列表上方，可以组合多个维度快速缩小范围。

## 筛选维度

- **文件类型** — 全部、图片、视频、文档
- **标签** — 下拉选择，支持多选
- **主色** — 11 种预设颜色，按图片主色调筛选
- **关键词** — 搜索文件名、备注和来源中的文本
- **日期范围** — 按开始/结束日期筛选
- **文件大小** — 最小/最大值（MB）
- **评分** — 从 1 星到 5 星

激活的筛选条件会在面板上显示计数，一键清除所有筛选。

## 排序

按任意维度排序，支持升序和降序：

- 导入时间
- 创建时间
- 修改时间
- 名称
- 类型（扩展名）
- 文件大小

## 搜索框

顶部搜索框按关键词检索，作用范围包括文件名、备注和来源链接。
