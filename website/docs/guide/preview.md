# 预览与详情

## 详情面板

选中文件后，右侧详情面板显示文件信息和操作入口：

- **预览** — 图片、视频、PDF、PSD 和文本文件的缩略图或内容预览
- **元数据** — 文件名、类型、大小、尺寸、导入时间、修改时间
- **标签** — 添加和编辑标签
- **评分** — 1-5 星评分，点击即可设置
- **备注** — 自由文本备注
- **来源链接** — 记录素材来源网址
- **颜色分布** — 图片主色的占比分布条
- **视频播放** — 详情面板内直接播放

## 全屏预览

双击文件或按快捷键进入全屏预览模式：

- **键盘导航** — 上/下键切换文件
- **缩放平移** — 触控板捏合缩放和平移
- **上下文菜单** — 右键提供打开、在 Finder 中显示、复制等操作

## 其他操作

在文件上右键，可以：

- 在 Finder / 资源管理器中显示
- 用默认应用打开
- 复制图片到剪贴板
- 导出到其他位置
- 重命名文件
