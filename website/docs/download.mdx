# 下载

请前往 GitHub Release 下载最新版本：

<a href="https://github.com/zihuv/shiguang/releases/latest" target="_blank" rel="noreferrer">打开最新 Release</a>

## 安装包

| 系统 | 建议下载 |
| --- | --- |
| Windows | `shiguang_*_x64.exe` |
| macOS | `shiguang_*_arm64.dmg` |
| Linux | `shiguang_*_x86_64.AppImage` 或 `shiguang_*_amd64.deb` |

macOS 版本未签名时，首次打开可能需要在系统设置中允许运行。
